# WpfGorgeousThemeSwitch
Wpf 精致的白天/黑夜主题切换开关控件

A exquisite day/night theme switch control for Wpf.

不使用图片资源文件，无第三方库依赖，矢量可缩放的控件样式，使用时只需为ToggleButton引用Style。
